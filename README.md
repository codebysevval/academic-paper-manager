# Academic Paper Manager

A simple backend application developed with Spring Boot 
to manage academic papers and related notes.

## Features
- Create and list academic papers
- Store basic metadata (title, authors, year, field)
- RESTful API design

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- 
## API Endpoints
- POST /papers
- GET /papers
- GET /papers/{id}
- DELETE /papers/{id}
